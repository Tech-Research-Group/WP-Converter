			</subpara1>
			<subpara1>
				<title>Explanations of Columns in Expendable/Durable Items List</title>
				<para>Column (1) Item No. This number is assigned to entry in list and is referenced in narrative instructions to identify item (e.g., Use brake fluid (WP 0098, Item 5)).</para>
				<para>Column (2) Level. This column identifies lowest level of maintenance that requires listed item:<randlist>
						<item>C - Crew</item>
					</randlist>
				</para>
				<para>Column (3) National Stock Number (NSN). This is NSN assigned to item which you can use to requisition it.</para>
				<para>Column (4) Item Name, Description, Part Number/(CAGEC). This column provides other information you need to identify item. Last line below description is part number and Commercial and Government Entity Code (CAGEC) (in parentheses)</para>
				<para>Column (5) U/I. Unit of Issue (U/I) code shows physical measurement or count of an item, such as gallon, dozen, gross, etc.</para>
			</subpara1>
